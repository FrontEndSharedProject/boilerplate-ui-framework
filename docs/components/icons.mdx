## 文件管理

字体图标依然使用 [IcoMoon](https://icomoon.io/) 生成,

对应存放目录为 `static/fonts` 下, 每次只需要将从 IcoMoon 上下载的 zip 包直接覆盖掉 `static/fonts` 中的内容即可
