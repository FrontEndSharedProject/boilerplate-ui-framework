## React 函数组件渲染

:::tip

在查看源代码之前, 请确保你对 函数组件 与 class 组件的区别有所了解

[https://zh-hans.reactjs.org/docs/components-and-props.html#function-and-class-components](https://zh-hans.reactjs.org/docs/components-and-props.html#function-and-class-components)

:::

<!---
你不应该在 mdx 文件中写 React 的组件
该文件应该只是用于 调用 React 组件
在 src/components 这个文件下编写你需要渲染给用户看的 React 组件, 然后再此处调用
此处这样写只是一个例子
-->

import { Card, CardData } from "@/components/Card";
const cardItem = CardData[0];

<section className="tw">
  <Card
    title={cardItem.title}
    info={cardItem.info}
    poster={cardItem.poster}
    tags={cardItem.tags}
  />
</section>
