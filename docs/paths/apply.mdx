```html
<button
  class="transform active:scale-95 bg-blue-500 hover:bg-blue-400 text-white px-16 py-6 rounded-lg font-bold tracking-widest w-full"
>
  <div class="flex justify-center items-center relative">
    <div class="button-copy pl-2 leading-none uppercase">DOWNLOAD</div>
  </div>
</button>
```

上面这么一个简单的按钮, 就要花费我们写这么一长串的 class, 那么如果后期需要变更一个样式, 这样就需要到各个项目中去修改 class 名, 这无疑是灾难性的问题

所以应该尽量使用 `@apply` 参考文档[https://tailwindcss.com/docs/extracting-components#extracting-component-classes-with-apply](https://tailwindcss.com/docs/extracting-components#extracting-component-classes-with-apply)

```scss
.btn-indigo {
  @apply py-2 px-4 bg-indigo-500 text-white font-semibold rounded-lg shadow-md hover:bg-indigo-700 focus:outline-none focus:ring-2 focus:ring-indigo-400 focus:ring-opacity-75;
}
```
