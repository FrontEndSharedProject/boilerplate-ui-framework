## 网站配置 docusaurus

想要配置网站, 请修改 `docusaurus.config.js`

[网站配置](https://docusaurus.io/zh-CN/docs/api/docusaurus-config)
[主题配置](https://docusaurus.io/zh-CN/docs/api/docusaurus-config)

## Tailwind 配置

请修改 `tailwind.config.js`

[https://tailwindcss.com/docs/configuration#app](https://tailwindcss.com/docs/configuration#app)

## typescript 配置

请修改 `tsconfig.js`

[https://www.typescriptlang.org/docs/handbook/tsconfig-json.html](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html)

## webpack 配置

docusaurus 未提供直接配置 webpack 的选项, 因此需要自己写插件进行配置, 请参考

[https://docusaurus.io/zh-CN/docs/using-plugins#creating-plugins](https://docusaurus.io/zh-CN/docs/using-plugins#creating-plugins)
