## 例子

<section className="tw">
  <div className="border-2 w-[50%] line-clamp-3 mb-4">
    To control the letter spacing of an element at a specific breakpoint, add a: prefix to any existing letter spacing utility. For example, use md:tracking-wide to apply the tracking-wide utility at only medium screen sizes and above.
  </div>
</section>


## 使用

[https://github.com/tailwindlabs/tailwindcss-line-clamp](https://github.com/tailwindlabs/tailwindcss-line-clamp)

```html
  <div className="line-clamp-3">
    To control the letter spacing of an element at a specific breakpoint, add a: prefix to any existing letter spacing utility. For example, use md:tracking-wide to apply the tracking-wide utility at only medium screen sizes and above.
  </div>
```

字体图标依然使用 [IcoMoon](https://icomoon.io/) 生成,

对应存放目录为 `static/fonts` 下, 每次只需要将从 IcoMoon 上下载的 zip 包直接覆盖掉 `static/fonts` 中的内容即可
